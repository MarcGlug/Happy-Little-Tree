# Happy Little Tree

Happy Little Tree est une web-app sous Vue.js qui permet de dessiner est sauvegarder vos oeuvres 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
